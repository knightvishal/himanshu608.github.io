HTML TAGs

TAG NAMES WHAT THEY DO???
<p> to write a paragraph
<h1>to<h6> to write headings as required
<b> to bold the text
<u> to underline text
<i> to italicize the text
<marquee> to move the text in a specified direction
<mark> to mark specific text
<sub> to subscript text
<sup> to superscript text
<strong> to provide strong importance to a specific text
<delete> to delete specified text
<address> while mentioning address
<q> to quote some phrase
<pre> to print the text as it is as mentioned by you in html file
<bdo> to print the text as mirror image
<hr> to make a horizontal line
<br> to jump from a line and go to another one
<table> to create a table
<!--...--> to put comments
<title> to define title of a document
<body> to create body of webpage
<head> to contain title, metadata and internal styling
<!DOCTYPE> to define the type of document
